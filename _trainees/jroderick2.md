### Biography and Interests
I am a first year graduate student in the CMS group at the University of Wisconsin-Madison.

### Project

I am working with Fermilab Scientist Dr. Scarlet Norberg on testing alternative storage systems for CMS at Fermilab. Currently,  two main filesystems are in use - T1 is a dCache storage system used for storage of production data and T3 is an EOS storage system which stores data used for analyses. The necessity of maintaining two different systems has recently come into question given the two systems have similar requirements. The goal of my project is to evaluate three options for new filesystems to be used to store both CMS analysis and production data at Fermilab. There are many requirements the new filesystems must meet. Most importantly, the new system must support currently used data access protocols (XRootD, HTTPS/WebDAV) and be highly scalable to keep up with the increased storage demands expected with the HL-LHC upgrade. I will be adapting both production and analysis workflows to run on the three different test systems and collect data on the performance of each system. This information can be used to help scientists at Fermilab make an informed decision on which new filesystem to implement.
